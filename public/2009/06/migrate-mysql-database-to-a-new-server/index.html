Migrate MySQL Database to a New Server

Jun 8th, 2009

Опять-таки огромная благодарность сайту shell -fu

Следующая команда скопирует базу данных old_db с локального хоста в new_db на указанном удаленном хосте:

    mysqldump --add-drop-table --extended-insert --force \
    --log-error=err.log -u[user] -p[pass] old_db | ssh -C user@host "mysql -u[user] -p[pass] new_db"
